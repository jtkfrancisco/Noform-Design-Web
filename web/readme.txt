Instructions on how to use the program properly.

First run the following command to install the necessary packages:
yarn install

Terminal 1 run the following command to watch the tailwindcss file:
npx @tailwindcss/cli -i ./src/assets/css/main-input.css -o ./src/assets/css/main-output.css --watch

Terminal 2 run the following command to start the program:
yarn start

To build the program run the following command:
yarn build
